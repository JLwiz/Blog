CSS Syntax and External Stylesheets

09/02/2021

The creation of CSS files provide alot of functionality and formatting for html files, and are a core actor in webpage development. Through some experimenting and practice, the CSS language provides a plethora of features important to any webpage - such as: color coding elements, margin sizing, and even gradiant color generation. Overall, the CSS files and syntax for webpage customization are easy to follow - much easier than html. To start of with using CSS to apply different formats to standard html, you must first select the html element(s) that you wish to apply some format to. Once the element is selected, by specifying with curly brackets afterwards (Element{}), properties may be applied to within the brackets to manipulate the formatting of the html. The property that is selected is followed by a colon and then a modifying value, as can be shown in the example below.

CSS code can be quick to make the html code look cluttered. To maintain some readability of html, it is more beneficial to link a .css file for general formatting. This is done through the link element. By specifying the relationship to the stylesheet (rel="stylesheet"), and the hypertext reference (href="docName.css"), this can be overcome. Though the use of external style sheets provide a lot of flexability in how CSS affects the html code; CSS stylesheets still can have draw backs. External style sheets can also cause the load speed of the connected webserver to slow due to the increase in download times for due to the added file.

Javascript is an object-oriented language that implements interactive elements into a web page. To link javascript into an html file, you must first implement a <script><script/> tag; unlike css styleguides, as discussed in the last post, a source must be declared within the <script> part of the script tag. This declaration is done by specifying the source then the file name like so:

<script src="scriptname.js"><script/>

To get started on writing the first javascript file for this web page is not all that difficult. In order to create a variable in javascript you must first declare the type var - javascript will interpret the variable type based of what the assignment is. Javascript uses functions to process calls from html files, which allow for verbose scripting calls. Next, call the .getHours(), .getMinutes(), and .getSeconds() functions on the Date object by assigning them new variables.
